# 포트폴리오 페이지

## 🌈 프로젝트 소개

개인 포트폴리오 소개를 위한 페이지 입니다.

## ⚙️ 프로젝트 설치 및 실행 방법

```Bash
# 프로젝트 설치
yarn install

# 프로젝트 실행
yarn start
```

## 🪃 Skills

### Client

- React
- React Router Dom
- Context API
- React Hooks
  - useState
  - useEffect
  - useContext
- Style
  - SCSS

### Database

- Firebase
  - Authentication
  - Firestore Database
  - Storage

### Server

- Firebase
  - Hosting
